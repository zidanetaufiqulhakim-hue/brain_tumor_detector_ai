About The Model

1. Model Used

Architecture: Tuned Xception Model (CNN-based)
The Xception architecture was fine-tuned on brain MRI data to support multi-class tumor classification with a focus on recall and interpretability.

2. Supported Classes

  • Healthy
  • Pituitary Tumor
  • Glioma
  • Meningioma
